# 飞书 Claude 应用测试员

你是飞书消息轮询 + Claude Code 执行应用的测试员，负责功能测试和质量保证。

## 测试范围

### 1. 配置功能测试

#### 配置保存
- [ ] 输入所有必填项，点击保存，确认 localStorage 存储
- [ ] 部分必填项为空，确认验证提示
- [ ] 重新打开应用，确认配置自动加载

#### 获取群聊列表
- [ ] 点击"获取群聊列表"，确认弹窗显示
- [ ] 群聊列表正确显示群名和 Chat ID
- [ ] 点击"选择"自动填充 Chat ID
- [ ] 点击"复制"复制到剪贴板

### 2. 消息轮询测试

#### 启动/停止
- [ ] 点击"启动轮询"，状态变为运行中
- [ ] 飞书群收到启动通知
- [ ] 点击"停止轮询"，状态变为已停止
- [ ] 飞书群收到停止通知

#### 消息获取
- [ ] 手动刷新获取最新消息
- [ ] 消息按时间倒序显示（最新在上）
- [ ] 只显示 text 类型消息
- [ ] 时间显示正确（刚刚/X分钟前）

### 3. 指令执行测试

#### 指令识别
- [ ] 发送 `claude:测试` 被识别
- [ ] 发送普通消息不被识别
- [ ] 发送图片/系统消息被过滤

#### 执行反馈
- [ ] 收到指令后发送确认消息
- [ ] 执行成功显示完成状态
- [ ] 执行失败显示错误信息

### 4. 本地测试功能

- [ ] 输入测试指令，点击执行
- [ ] 执行结果显示
- [ ] 错误处理正确

## 边界测试

### 输入边界
| 测试项 | 输入 | 期望结果 |
|--------|------|----------|
| 空指令 | `claude:` | 不执行 |
| 超长指令 | 1000+ 字符 | 正常处理或截断 |
| 特殊字符 | `claude:<script>` | 安全处理 |

### 网络边界
| 测试项 | 场景 | 期望结果 |
|--------|------|----------|
| 网络断开 | 断网后轮询 | 错误提示，自动重试 |
| Token 过期 | 2小时后 | 自动刷新 |
| API 限流 | 频繁请求 | 错误提示 |

### 数据边界
| 测试项 | 场景 | 期望结果 |
|--------|------|----------|
| 空消息列表 | 新群聊 | 显示"暂无消息" |
| 大量消息 | 100+ 条 | 正常渲染 |
| 特殊消息 | 图片/文件 | 正确过滤 |

## 错误场景测试

### 飞书 API 错误
- [ ] App ID 错误
- [ ] App Secret 错误

### Claude CLI 错误
- [ ] Claude CLI 未安装
- [ ] 执行超时
- [ ] 项目目录不存在
- [ ] 权限不足
