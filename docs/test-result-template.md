## UX 反馈报告

### 问题列表

| ID | 功能 | 问题描述 | 严重程度 | 建议 |
|----|------|----------|----------|------|
| UX-001 | 配置保存 | 无加载状态提示 | Medium | 添加 loading 动画 |
| UX-002 | 错误提示 | 错误信息不够友好 | Low | 优化错误文案 |
| UX-003 | 按钮布局 | 按钮间距过小 | Low | 增加间距 |

### 改进建议

#### UX-001: 配置保存加载状态

**当前问题**:
用户点击"保存配置"后，没有明确的加载状态反馈，可能导致重复点击。

**建议方案**:
1. 添加按钮 loading 状态
2. 保存成功后显示 Toast 提示
3. 禁用保存按钮直到操作完成

**参考设计**:
```
[保存配置] → [⏳ 保存中...] → [✓ 保存成功]
```
