# Icons Directory

This directory should contain application icons for different platforms.

Required icons:
- 32x32.png - Windows taskbar icon
- 128x128.png - Windows application icon
- 128x128@2x.png - macOS Retina icon
- icon.icns - macOS icon bundle
- icon.ico - Windows icon

You can generate these icons using the Tauri CLI:
```bash
npm run tauri icon /path/to/your/source-icon.png
```
