## 4. 关键决策

### 4.1 图片上传实现
使用 Tauri 的 `fetch` API 直接上传到飞书服务器，避免 CORS 问题。

### 4.2 测试方案
使用 tauri-driver + WebdriverIO 进行 E2E 测试，完全自动化，无需人工干预。
